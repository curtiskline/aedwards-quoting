# Ground Truth Extraction

Extract structured data from Allen Edwards quote PDFs using Claude's vision capabilities.

## Quick Start: Spot Check (10 random PDFs)

```bash
# Generate the extraction prompt
python tools/spot_check_ground_truth.py

# Then paste the output into Claude Code:
claude --dangerously-skip-permissions -p "$(python tools/spot_check_ground_truth.py)"
```

Options:
```bash
python tools/spot_check_ground_truth.py --count 5     # fewer PDFs
python tools/spot_check_ground_truth.py --seed 42     # reproducible selection
python tools/spot_check_ground_truth.py --list         # just list selected files
```

## Prerequisites

- Claude Code CLI installed (`claude` command available)
- Anthropic API key set (`ANTHROPIC_API_KEY` environment variable)
- Quote PDFs in `data/curtis-emails/attachments-jamee-sent/`
- Manifest generated by `tools/match_attachments.py` at `data/test-corpus/manifest.json`

## What It Does

1. Picks N random quote PDFs from the attachment manifest
2. Generates a prompt that tells Claude to read each PDF and extract structured JSON
3. Claude reads each PDF visually and writes a `.json` file to `data/test-corpus/ground-truth/`

## Output Schema

Each JSON file contains:
- `customer_name` — company from Bill To
- `contact_name`, `contact_email`, `contact_phone` — from Requested By
- `ship_to` — company, attention, street, city, state, postal_code, country
- `po_number`, `quote_number`
- `line_items[]` — part_number, description, quantity, unit_price, total
- `subtotal`, `shipping_amount`, `tax_amount`, `total`
- `_meta` — source_pdf, extraction_status, extraction_method, parsed_at

## Cost Estimate

Each PDF extraction uses ~16k tokens (~$0.01-0.05 depending on model).
10 PDFs ≈ $0.10-0.50. Full corpus (1,271 PDFs) ≈ $15-60.
